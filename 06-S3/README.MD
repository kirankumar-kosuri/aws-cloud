# S3[Simple Storage Service]
============================
* S3 is nothing but it is a simple storage service provided by AWS
* S3 is also known as Globally Accessiable Service
* In Orginazation there will be huge amount of data to store that data than S3 comes into picture 
* In S3 we can store anything like Photos,Videos,Csv files and more
* In S3 there is no limit for storage we can store huge amount of data upto 5TB if more than that we have to breake the bucket 
* In S3 we can create buckets and in that buckets we can store what we want
* As a DevOps Engineer we store data in S3 like Files,Application Logs,Excel Sheets and Databases
* What data that we stored in S3 that data we can access through HTTP Protocol can acccess in the S3 bucket from anywhere from the world 
* The S3 bucket name should be unique across the Aws Accounts 

# Advantage's of S3
===================
Durability and availability: S3 provides high durability and availability for your data.
Scalability: You can store and retrieve any amount of data without worrying about capacity constraints.
Security: S3 offers multiple security features such as encryption, access control, and audit logging.
Performance: S3 is designed to deliver high performance for data retrieval and storage operations.
Cost-effective: S3 offers cost-effective storage options and pricing models based on your usage patterns.
